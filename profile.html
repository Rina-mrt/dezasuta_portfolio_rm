Profile

自己紹介

"もりた りな" のポートフォリオサイトをご覧いただき、ありがとうございます。

私は現在、冊子の編集業務をしています。付随して、取材と記事の執筆や、リーフレットの作成なども行っています。この度、デザインの勉強を兼ねて、スキルアップのためにWebデザイナーを育成する講座を受講しました。

毎日勉強していく中でWebサイト制作のおもしろさに触れ、学んだことを仕事として取り組みたいと考えるようになりました。このポートフォリオサイトが、そのための第一歩となればいいなと思います。

まだまだ勉強したいことは尽きません。どなたかのお役に立てるよう、これからも多方面で勉強を続け、日々ステップアップしていきます！

プロフィール
札幌市出身、埼玉県育ち。
学生時代は心理学を専攻、子どもの発達支援や就学児デイサービスのボランティアをしていました。卒業後は秘書業務を経て、総務・人事業務を経験。その後しばらくは子育てに専念していましたが、子どもの就学を機に現職に就きました。
使用できるソフト：Illustrator、InDesign、Photoshop、Visual Studio Code
わたしの強み
論理的思考 と 批判的思考

このことは特に、卒業論文と修士論文のための研究活動を通して培ってきました。サイト本来の目的を検証したり、最適なデザインを検討する上で役立てることができると思います。

「聞く」を重視したコミュニケーション

最適なサポートを提供するために、支援や業務の中で「相手の立場や状況を把握する」ことが求められてきました。お客様の要望を正確に聞くだけではなく、言語化できていないニーズを引き出し、本当に実現したいことを叶えられるご提案をできたらと思います。

数値を正確に扱うこと

研究活動では統計分析のために様々な指標を扱い、業務では旅費の計算・支給を担当していました。数値を丁寧に扱う必要のある、地道で細かい作業は得意です。大前提となる正確性についても、精度の高い仕事を完遂できるよう心がけています。

調整力

他機関・他部署との連携や、多岐にわたる業務を期日を守って効率的にこなすこと、最良の結果を出すためにどうするかということが常に求められてきました。他者とのスムーズな業務遂行、スケジュール管理など様々な場面で調整力は求められますが、私の得意なことなので、これまでの経験を活かしていきたいと思います。

文章を書くこと

前職では経費削減に向けた提案書や、新しい制度の運用について規約や周知文書の作成を行ってきました。現在はいかに伝えたい内容を届けるか、文字数と的確な表現を検討しながら編集や執筆をしています。書くことが大好きなので、サイトの制作においても寄与することができると考えます。
